## Is this for me?

- Your PC takes 5 minutes to boot? **Yes.**
- Right-click menu appears after 10 seconds? **Absolutely yes.**
- You have 47 Korean banking security programs? **Oh god yes.**
- Your C: drive is mysteriously full? **We got you.**
- Explorer freezes when you open a folder? **Say no more.**

## One command. That's it.

```bash
npx pc-detox
```

No install. No signup. No BS. Just run it.

## Features

| # | Feature | What it does |
|---|---------|-------------|
| 1 | **PC Detox Scan** | Find & remove apps you don't need |
| 2 | **Startup Manager** | Speed up boot time |
| 3 | **Process Monitor** | See what's running right now |
| 4 | **One-Click Cleanup** | Junk, cache, registry — all at once |

## Safety First

PC-Detox will **never** do anything without asking you first.

- Every destructive action requires confirmation
- Risk levels shown for each action (SAFE / LOW RISK / HIGH RISK)
- `--dry-run` mode to preview without touching anything
- System essentials are protected and can't be removed
- No data leaves your computer. Ever.
